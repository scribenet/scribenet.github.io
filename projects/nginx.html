Nginx & Plugins

A collection of scripts we use to build our Nginx PPA, as well as slight changes made to select modules required for compatibility.

This group of projects is also available via our internal Git management solution at: https://scribe.software/code/projects/NGX

  • Nginx Fancy Index Theme

    A stylist and simple theme for the Nginx fancyindex module.

    https://scribe.software/code/projects/NGX/repos/nginx-fancy-index-theme/clone
  • Nginx HTTP Extended Status Module

    This module provides extended Nginx statistics. It requires patching the Nginx source and as such a patch file must be made for each new Nginx release.

    https://scribe.software/code/projects/NGX/repos/nginx-http-extended-status-module/clone
  • Nginx Mainline Package Builder

    This script builds a set of Nginx packages against the Ubuntu Nginx Mainline Branch, currently at version 1.7.8, with an emphasis on producing a streamlined final binary that includes/excluded configuration flags and adds a short list of modules used internally by Scribe Inc.â€”for both our own websites and those we administer.

    https://scribe.software/code/projects/NGX/repos/nginx-mainline-package-builder/clone
