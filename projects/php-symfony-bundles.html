PHP Libraries

Collection of open-source PHP Library components fully compatible with Composer.

This group of projects is also available via our internal Git management solution at: https://scribe.software/code/projects/PHPL

  • Jabiru Markdown Bundle

    Bundle to intwgrate the Jabiru markdown parser into Symfony.

    https://scribe.software/code/projects/PHPS/repos/jabiru-markdown-bundle/clone
  • Cache Bundle

    Simplistic cache abstraction layer (for APC only at this time) for Symfony2.

    https://scribe.software/code/projects/PHPS/repos/scribe-cache-bundle/clone
  • Clockwork Bundle

    Symfony bundle to handle interaction with the ClockworkSMS.com API for SMS messaging.

    https://scribe.software/code/projects/PHPS/repos/scribe-clockwork-bundle/clone
  • File Uploader Bundle

    Simple multi-uploader based off of the work by Punkave and Blueimp.

    https://scribe.software/code/projects/PHPS/repos/scribe-file-uploader-bundle/clone
  • Scribe Foundation Bundle

    The base Symfony framework, dependencies, and shared utilities used for Scribe Inc. internal Symfony projects. We also maintain a collection of open-source projects related to Symfony, Sublime Text, jQuery, and more.

    https://scribe.software/code/projects/PHPS/repos/scribe-foundation-bundle/clone
  • GitHub API Bundle

    Object oriented PHP5 client for GitHub's API (v3).

    https://scribe.software/code/projects/PHPS/repos/scribe-github-api-bundle/clone
  • GitLab API Bundle

    Object oriented PHP5 client for GitLab's API.

    https://scribe.software/code/projects/PHPS/repos/scribe-gitlab-api-bundle/clone
  • Image Magick Bundle

    Symfony bundle to provide an abstraction layer over the PHP IMagick library with higher-level, reusable functionality.

    https://scribe.software/code/projects/PHPS/repos/scribe-image-magick-bundle/clone
  • Markdown Bundle

    This bundle lets you easily make use of the markdown parser inside your Symfony applications.

    https://scribe.software/code/projects/PHPS/repos/scribe-markdown-bundle/clone
  • Pheanstalk Bundle

    Symfony2 Bundle that provides a pheanstalk integration.

    https://scribe.software/code/projects/PHPS/repos/scribe-pheanstalk-bundle/clone
  • PHP Spec Code Coverage Bundle

    This project was forked from henrikbjorn/PhpSpecCodeCoverageExtension to customize some of the inner workings of this project to better align with the workflow I use both at home and when I work for Scribe Inc. Additional information as to what that means will be added shortly.

    https://scribe.software/code/projects/PHPS/repos/scribe-php-spec-code-coverage-bundle/clone
  • Pygments Bundle

    Provides (X)HTML rendering of Markdown similar to GFM (Github flavored markdown), including syntax-highted fenced code blocks.

    https://scribe.software/code/projects/PHPS/repos/scribe-pygments-bundle/clone
  • Resque Bundle

    The BCC resque bundle provides integration of php-resque to Symfony2. It is inspired from resque, a Redis-backed Ruby library for creating background jobs, placing them on multiple queues, and processing them later.

    https://scribe.software/code/projects/PHPS/repos/scribe-resque-bundle/clone
  • Slack Bundle

    This Bundle will integrate Slack Team-Communication-Software into your Symfony2 Project and in the future will deliver a set of Project-Management-Tools to use between your Symfony-Project and Slack.

    https://scribe.software/code/projects/PHPS/repos/scribe-slack-bundle/clone
  • Stripe Bundle

    Symfony bundle to handle interaction with the Stripe.com payments API.

    https://scribe.software/code/projects/PHPS/repos/scribe-stripe-bundle/clone
  • SWiM Bundle

    A Scribe Simple Wiki Markup bundle for Symfony.

    https://scribe.software/code/projects/PHPS/repos/scribe-swim-bundle/clone
  • Syslog Bundle

    Scribe Syslog to common entity facilities bundle.

    https://scribe.software/code/projects/PHPS/repos/scribe-syslog-bundle/clone
  • Two-Factor Auth Bundle

    This Symfony2 bundle provides two-factor authentication for your website.

    https://scribe.software/code/projects/PHPS/repos/scribe-two-factor-auth-bundle/clone
